# PODs Lineup / Available Coverage

A real-time lineup and coverage tracking tool for charge nurses.

**Live app:** https://tleebraun.github.io/PODs-Lineup-Available-Coverage/

## Using in Microsoft Teams

No custom app installation or IT approval required. Use the built-in **Website tab**:

1. Open the Teams channel or chat where you want to pin the tool
2. Click **+** (Add a tab) at the top
3. Choose **Website**
4. Paste the URL: `https://tleebraun.github.io/PODs-Lineup-Available-Coverage/`
5. Name it "PODs Lineup" and click **Save**

The app automatically detects when it's running inside Teams, applies the Teams theme (light/dark/high-contrast), and pre-fills your display name from your Teams identity.

The **📌 Pin to Teams** button in the app header also provides this URL and instructions when viewed in a regular browser.

## Setup

On first load, configure your Firebase Realtime Database credentials in the Setup card. All users on the same "room code" share live state.
